# Module 29: Docker and Kubernetes Deployment

## Understand

Docker and Kubernetes deployment packages API and web apps into containers and deploys them with services, ingress, configuration, secrets, and autoscaling.

## Deliverables

- API Dockerfile
- Web Dockerfile
- API deployment/service/config/secret example/HPA
- Web deployment/service/HPA
- Ingress
- Observability config

## Security Rules

- Do not commit production secrets.
- Use Kubernetes Secrets or external secret managers.
- TLS termination and domain mapping must be configured per environment.

## Tests

Run `npm run test` from the repository root.
