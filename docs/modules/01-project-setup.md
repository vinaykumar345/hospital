# Module 1: Project Setup

## Understand

Project setup establishes the monorepo, app boundaries, shared package, documentation, infrastructure placeholders, verification scripts, and baseline safety requirements.

## Plan

- Backend: scaffold NestJS API package with health route structure.
- Frontend: scaffold React package with role-aware shell placeholder.
- Mobile: scaffold Flutter package metadata and first screen.
- Database: document schema conventions; migrations begin in authentication and tenancy modules.
- AI: centralize the mandatory AI review disclaimer.
- Security: add environment samples, ignore secrets, and document baseline controls.
- Tests: add setup verification and package-level smoke tests.

## Verification

Run `npm run verify` from the repository root.
